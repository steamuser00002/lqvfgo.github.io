跨版本升级deepin15.11到deepinv20

发表于 2020-09-26 分类于 编程
  1. 将15.11官方源替换成20官方源（/etc/apt/sources.list）

deb [by-hash=force] https://community-packages.deepin.com/deepin/ apricot main contrib non-free

  1. 新增20官方商店源（/etc/apt/sources.list.d/appstore.list ），无list文件创建后添加：

deb https://community-store-packages.deepin.com/appstore eagle appstore

  1. 命令行更新

sudo apt-get update -y && sudo apt-get upgrade -y

  1. 重启执行

sudo apt-get update -y && sudo apt-get upgrade -y
